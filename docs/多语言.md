# 多语言

## 第一步 确保main.js中已经引入多语言包

```js
import i18n from '@/i18n/i18n'
new Vue({
  i18n,
}).$mount('#app')
```

## 第二步 在多语言包中设置对应的变量并写上对应语言

比如在 i18n/lang/en-us中添加=》titleCont

```js
import enUS from 'vant/lib/locale/lang/en-US'
export default {
  ...enUS,
  titleCont: 'titleCont'
}
```

## 第三步 使用多语言变量

```html
<div>{{$t('titleCont')}}</div>
```
